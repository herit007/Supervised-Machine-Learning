# 🏠 House Price Prediction Using Regression Models

## 📌 Project Overview

This project focuses on predicting house prices using different regression techniques in Machine Learning. The project follows a complete workflow starting from data understanding and preprocessing to model training, evaluation, and performance comparison.

The main objective is to analyze how different regression models perform in predicting house prices and understand concepts such as Gradient Descent, Bias-Variance Trade-off, and model evaluation metrics.

---

# 🎯 Project Objectives

* Understand the House Price dataset
* Perform data cleaning and preprocessing
* Visualize relationships between variables
* Build Simple Linear Regression models
* Build Multiple Linear Regression models
* Apply Polynomial Regression
* Implement Gradient Descent techniques
* Evaluate model performance
* Compare different regression approaches
