# 📊 Customer Churn Prediction Using Logistic Regression

A Machine Learning classification project that predicts whether a customer is likely to churn (leave the company) or remain with the company using Logistic Regression and evaluates model performance using various classification metrics.

---

## 📖 Overview

Customer churn prediction is one of the most important applications of machine learning in customer relationship management. Businesses use churn prediction models to identify customers who are likely to discontinue their services and take proactive measures to retain them.

This project uses a Logistic Regression classification model to analyze customer demographic information, subscription details, spending behavior, and service interaction data to predict whether a customer is likely to churn.

---

## 📊 Dataset Information

### Features

* Customer ID
* Age
* Gender
* Tenure
* Usage Frequency
* Support Calls
* Payment Delay
* Subscription Type
* Contract Length
* Total Spend
* Last Interaction

### 🎯 Target Variable

**Churn**

* 0 → Customer Stays
* 1 → Customer Churns
